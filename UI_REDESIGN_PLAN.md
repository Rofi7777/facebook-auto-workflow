# UI 重构计划 - Googa Ai Hub

## 设计目标
1. **简化流程**：从3个独立按钮改为更直观的步骤式流程
2. **现代化设计**：使用卡片式布局、渐变效果、微交互
3. **清晰的信息层次**：使用步骤指示器、进度条、状态反馈
4. **响应式设计**：适配各种屏幕尺寸

## 新的工作流程

### 步骤1: 产品信息输入（简化表单）
- 产品名称（必填）
- 产品图片上传（拖拽上传，支持多图预览）
- 目标平台选择（图标化选择）
- 语言选择

### 步骤2: AI 分析（自动触发）
- 上传后自动开始分析
- 显示分析进度
- 实时显示分析结果

### 步骤3: 内容生成（一键生成）
- 显示所有平台的预览卡片
- 每个卡片有独立的"生成图片"按钮
- 实时显示生成进度

## UI 组件设计

### 1. 步骤指示器
```
[1] 产品信息 → [2] AI 分析 → [3] 内容生成
```

### 2. 卡片式布局
- 每个平台一个卡片
- 卡片内包含：平台图标、内容预览、操作按钮

### 3. 图片上传区域
- 拖拽上传
- 多图预览网格
- 删除功能

### 4. 平台选择
- 图标化按钮
- 选中状态明显
- 支持全选/取消全选

## 技术实现
- 使用 CSS Grid 和 Flexbox
- 添加微动画和过渡效果
- 使用 CSS 变量统一主题色
- 响应式断点设计
